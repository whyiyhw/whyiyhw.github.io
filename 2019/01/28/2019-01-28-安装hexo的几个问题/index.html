安装 hexo 静态 blog 的一些问题

npm 和 yarn 的混用

- hexo new [layout] <title>  发布失效
    - 全局安装 hexo 的时候失效 npm 有毒 
    - yarn add hexo -S
- yarn add hexo-deployer-git -S  发布工具
- yarn add  hexo-renderer-marked  -S
- yarn add hexo-server -S 本地起服务  hexo server -p 5000

配置Deployment

- 本地记住了 git 账号 所以直接使用 这里空格有坑
- deploy:
    - type: git
    - repository: https://github.com/whyiyhw/whyiyhw.github.io.git
    - branch: master

发布

- hexo d -g 前面安装正常 发布就没问题
