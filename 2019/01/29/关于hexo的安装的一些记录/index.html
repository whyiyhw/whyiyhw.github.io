基础安装流程

  • 安装 hexo cnpm install hexo -g
    • hexo -v
  • 安装 hexo 命令行 cnpm install -g hexo-cli
    • cd test/
    • hexo init
    • hexo new post "关于hexo的安装的一些记录"
    • cnpm install hexo-server --save
    • hexo server
    • hexo generate --deploy
